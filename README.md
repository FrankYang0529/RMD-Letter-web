# RMD-Letter web

Node v6.10.0

## Setup

### MongoDB

macOS

```
brew install mongodb
```

### Redis

macOS

```
brew install redis
```

### nodemon

```
npm install nodemon -g
```

### npm packages

```
npm install
```

## Start Server

### MongoDB

```
sudo mongod
```

### Redis

```
redis-server
```

### Web Server

```
npm start
```
